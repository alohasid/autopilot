Автономна Місія БПЛА

Розробка скрипта керування коптером через SITL симулятор з використанням Dronekit. Зліт, політ за координатами та точна посадка.

⚠️

Критична вимога

Використовувати виключно RC Override! Режим: Stabilize. Точність посадки впливає на бал.

🛠️ Етапи підготовки

Натисніть на крок для відображення інструкцій та параметрів.

Завантажте та встановіть середовище для Windows або Ubuntu.

Перейти до завантаження ↗

Запустіть симулятор та встановіть наступні параметри вітру для створення складних умов керування:

  • SIM_WIND_SPD = 3
  • SIM_WIND_DIR = 30
  • SIM_WIND_TURB = 2
  • SIM_WIND_TURB_FREQ = 0.2

Бібліотека для керування дроном через Python.

Документація: dronekit.io

pip install dronekit
